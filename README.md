# The-Finals-Anti-Recoil
**Maybe detected wouldn't run on main account.**
## How to run
Open the "settings.ini" file using Notepad and set your sensitivity and FOV settings.\
Run "RunMe.ahk"
## Hotkeys
TTS should tell you what is selected
- F1 - Disabled
- F2 - AKM
- F3 - XP-54
- F4 - Lewis gun
- End - Closes Program\
